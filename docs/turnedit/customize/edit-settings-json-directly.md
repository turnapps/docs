# Edit Settings JSON directly
The TurnEdit settings is built with JSON. Here's how to edit settings JSON directly:
# Warning
- **Syntax error**: If JSON file have syntax error, occurs settings load error and loads default settings. It's highly recommended check JSON syntax manually.
- **Backup settings file**: It's highly recommended to backup JSON file.
## 1. Open a settings JSON file in any text editor
JSON file location: C:\Users\{username}\AppData\Roaming\TurnEdit\turnedit-settings.json
Open a settings file in explorer, or text editor.
## 2. Reload settings
1. If you opened settings window, close it.
2. Open settings.
3. Do nothing and click "OK" button.
