# How to fix settings load errors
If occur settings load error when TurnEdit startup, or opening settings screen, try below:
## Fix the settings file
If found JSON error, try below:
* Check that it doesn't contain syntax errors.
* Check a settings file is JSON properly.
### File encoding problem
* Check an encoding of settings file is UTF-8.
* Check a line break code of settings file is CRLF.
## Check a file is accessible
* Check a permission of settings file is granted.
## Others
If this problem continues, please post question to [forum](https://turnapps.f5.si/turnedit/the-forum/).
