# Minimum System requirements
## Operating system
- Minimum version: Windows 10 22H2 or greater
- .NET version: .NET is contained in the TurnEdit package; if install .NET manually, use .NET 9 or 10.
## Hardware
- CPU: 1GHz or greater
- Memory: Minimum 2GB or greater, recommends 4GB or greater
- Graphics: DirectX 9 or greater
- Disk free space(e.g. HDD, SSD): 400MB or greater
## Notes
TurnEdit uses rendering engine as DirectX. the rendering engine can't be changed.
